# C++ OpenGL开发的3D打飞机游戏

窗口库用的glfw

界面库用的dear-imgui

模型加载用的assimp

地图用FastNoise库生成的Berlin Noise

通过存储旋转矩阵，把pitch, yaw变化量叠加到旋转矩阵的方法，解决了万向节死锁的问题。
